Open Your Library

LibrAIry does not serve SMB, FTP, WebDAV, or any other file-sharing protocol. It only tells you where your files live so you can expose them with your NAS or operating system.

Host Paths

Library: {{ host_library_dir }} Inbox: {{ host_inbox_dir }} Quarantine: {{ host_quarantine_dir }}

UNRAID SMB

1. In UNRAID, open Shares and create or select the share that contains {{ host_library_dir }}. 2. Enable SMB export for that share. 3. From Windows, try \\TOWER\library or the share name you configured. 4. From macOS/Linux, try smb://TOWER/library or your server IP.

FTP / WebDAV

Use your normal UNRAID plugin, Docker app, NAS package, or OS service to expose {{ host_library_dir }}. Point that service at the host path above, not at LibrAIry's internal /data/library path.

Generic Systems

Share {{ host_library_dir }} with the file-sharing service you already trust on this machine. If you moved the host library path, update HOST_LIBRARY_DIR in the container environment and restart LibrAIry.
